# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in rove, please **do not open a public GitHub issue**. Public disclosure before a fix is available puts other users at risk.

Instead, report it privately:

- **Email:** isnielsen13@gmail.com

Please include a description of the issue, steps to reproduce it, and any relevant context. I'll aim to respond within a few days and will keep you updated as I work on a fix.

Thank you for helping keep rove safe.
